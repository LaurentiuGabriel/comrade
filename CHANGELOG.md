# Changelog

All notable changes to Comrade will be documented in this file.

## [0.1.0] - 2024-01-01

### Added
- Initial release of Comrade
- Electron + React desktop application
- Express API server
- Workspace management
- Session-based chat interface
- Skills system for customization
- Command-line interface
- Token-based authentication
- Audit logging
- CORS support for web clients
- Dark/light mode support

### Features
- Create and manage multiple workspaces
- Start AI-powered chat sessions
- Create, edit, and delete skills
- View session history
- Desktop notifications via Toast
- Collapsible sidebar navigation
- Responsive UI design
- Keyboard shortcuts (Enter to send, Shift+Enter for new line)

### Technical
- TypeScript throughout
- Redux Toolkit for state management
- CSS Modules for styling
- WebSocket support for real-time updates
- SQLite for local data storage
- Modular architecture with clear separation of concerns
